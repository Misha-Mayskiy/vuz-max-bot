# Вузуслуги - Frontend

Фронтенд часть мини-приложения "Вузуслуги" для мессенджера MAX.

## Технологии

- React 18
- TypeScript
- Vite
- MAX UI (@maxhub/max-ui)
- React Router
- Axios
- date-fns

## Функционал

- **Аутентификация**: Вход в систему через OAuth2
- **Расписание**: Просмотр и управление расписанием занятий
- **Управление данными**: CRUD операции для групп, преподавателей и предметов
- **Документы**: Заказ справок и документов
- **Общежитие**: Оплата проживания, заказ пропусков, техподдержка
- **Библиотека**: Заказ книг и доступ к электронным ресурсам
- **Деканат**: Запись на прием и подача заявлений
- **Финансы**: Оплата обучения и информация о стипендиях

## Структура проекта

```
src/
├── api/           # API клиент
├── contexts/      # React контексты (Auth)
├── components/    # Кастомные компоненты
├── pages/         # Страницы приложения
├── types/         # TypeScript типы
├── App.tsx        # Главный компонент с роутингом
├── main.tsx       # Точка входа
└── index.css      # Глобальные стили
```
